## 結構資料
### 提供下載

有時候會有很多勾勾，這時打開**開發者工具**，按**console**輸入：

```javascript
$("input[type='checkbox']").prop("checked", true);
```

就可幫你全部打勾

### html table
下載網頁成`.html`，至google搜尋: **html to csv**

或至[**這裡**](http://www.convertcsv.com/html-table-to-csv.htm)上傳html，該網頁會幫你抓出table，接著就能直接下載成`.csv`
